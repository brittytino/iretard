# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Provide actionable technical feedback.

## Unacceptable Behavior

- Harassment, abuse, threats, or discrimination.
- Personal attacks, trolling, or sustained disruption.
- Sharing private information without consent.

## Enforcement

Project maintainers may remove comments, reject contributions, or block participants for behavior that violates this Code of Conduct.

## Reporting

Open an issue with the label conduct or contact the maintainer through GitHub profile messaging.
