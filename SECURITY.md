# Security Policy

## Supported Versions

The latest main branch is supported for security fixes.

## Reporting a Vulnerability

Please report security issues privately using GitHub Security Advisories for this repository.

If advisories are unavailable, open a private communication request via the maintainer profile and avoid public disclosure.

Include:

- Steps to reproduce
- Impact assessment
- Suggested mitigation (if known)

## Disclosure

Do not post exploit details publicly until a fix is available.
